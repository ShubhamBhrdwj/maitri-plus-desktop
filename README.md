# Maitiri+ Android

Download the latest APK from **Releases**.

## 📦 Install
1. Open the **Releases** tab
2. Download the newest `Maitri+ Setup 1.0.0.exe`
3. Install it on your Windows Device  
   (Using the Setup Install the Application)

## ✅ Notes
- Works on Windows 10+
- Requires internet connection
